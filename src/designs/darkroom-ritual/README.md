# 方案一：暗房仪式派（Darkroom Ritual）

## 设计理念

模拟真实暗房操作的沉浸体验，将数字照片管理流程转化为一套充满仪式感的暗房操作序列。
深色为主调，琥珀色安全灯光晕点缀，胶片颗粒噪点覆盖，让每一次操作都像在真正的暗房里冲洗底片。

---

## 三步冲洗流程

### Step 01 · 挑选底片（FilmSelector）

- 底片条以「晾衣绳悬挂」形式展示，每条胶卷悬挂于绳上
- 每卷底片显示：挂片夹、齿孔、缩略帧色块、胶卷型号/ISO/日期标签
- 悬停底片条时有缓慢浮起动效，信息栏同步显示详情
- 支持「全部/加星标」过滤
- 路由：`/design/ritual`（默认进入此步骤）

### Step 02 · 调配显影液（ChemicalBath）

- 拟物滚筒拨盘：上下拖动选择机身/镜头/显影液，模拟老式机械选择器
- 拟物旋钮：拖动调节显影温度（15–30°C）和推拉曝光（-2 到 +2 stops）
- 配方摘要实时展示在底部确认栏
- 点击「注入显影液」触发 2 秒等待动画后进入下一步

### Step 03 · 放大台冲洗（EnlargerStation）

- 左侧胶片条：所有帧的缩略索引，有调整的帧显示橙色标记点
- 中央灯箱：照片呈现在发光灯箱台面上，亮度可通过顶部滑块调节
- 悬停照片：触发「对焦环」覆盖层，呈现完整的暗房式参数调整界面
- 调整项：曝光 EV、对比度、高光、阴影、色温、胶片颗粒感
- 调整实时反映在照片的 CSS filter 和颗粒叠层上
- 右侧信息栏：卷级信息、调整进度条、操作提示

## 样式特点

| 特性 | 实现方式 |
|------|----------|
| 深色背景 | `#050505` 主背景，`#060400` 次级面板 |
| 安全灯色调 | 琥珀色 `#ff9900`，暗绿色 `#2d5a27` 光晕 |
| 胶片颗粒 | SVG feTurbulence 噪点层，固定在视口上方 |
| 过渡动画 | `cubic-bezier(0.25, 0.46, 0.45, 0.94)` 缓慢柔和曲线 |
| 安全灯呼吸 | `safelight-pulse` keyframe，8s 周期 |
| 颗粒抖动 | `grain` keyframe（可选激活） |
| 字体 | `monospace` 等宽字体强化机械感 |

## 交互操作说明

- **底片条**：鼠标悬停浮起，点击进入显影流程
- **滚筒拨盘**：上下拖动滚动选项，或点击右侧箭头逐项切换
- **旋钮**：上下拖动旋转调节数值
- **放大台**：鼠标悬停照片，自动弹出对焦环调整面板
- **对焦环内滑块**：点击滑轨任意位置即可跳转数值
- **灯箱亮度**：顶部工具栏滑块实时调节
- **缩放**：顶部 +/- 按钮调节放大台视图比例
