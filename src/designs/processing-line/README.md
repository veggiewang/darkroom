# 方案二：效率流水线派（Processing Line）

路由：`/design/line`

## 设计理念

批量处理效率至上，键盘驱动，最大化信息密度，减少无效点击。整体采用白色/浅灰背景配高对比度黑色文字，蓝色（#3b82f6）作为唯一操作主色，视觉噪音极低。

## 核心流程

```
拖入文件夹 → 快速填写卷信息 → 时间轴键盘审阅 → 导出
```

四个阶段通过顶部阶段条（Stage Bar）时刻提示进度，不可逆操作前均有确认步骤。

## 各组件说明

### `BatchImporter.tsx`
- 大面积拖放区域，进入拖拽状态时有动画反馈
- 拖入后自动扫描，进度环形指示器
- 扫描结果摘要：文件夹名、检测文件数、可导入帧数、不支持格式提示
- 支持的格式：TIF/TIFF/JPG/JPEG/PNG/RAW/DNG/CR2/CR3/NEF/ARW

### `QuickFillForm.tsx`
- 按信息类型分区：胶卷、器材、日期、冲洗实验室
- 胶卷型号带自动补全下拉，选中自动填充 ISO 和胶卷类型
- **"从上次复制"**：一键复制上次设置的器材/胶卷信息，只保留当前日期
- **自动保存草稿**：所有改动实时写入 localStorage，刷新后恢复
- 推拉冲洗 ±2 档步进控件
- 折叠式"附加信息"区域减少默认视觉噪音
- 底部摘要标签实时预览已填内容

### `TimelineStrip.tsx`
- 仿胶片条样式（两侧齿孔装饰），深色背景与主区域形成分隔
- 缩略图宽 72px，保留足够辨识度
- 状态色点：灰色=待处理、蓝色=已审阅、黄色=已标记、绿色=已确认、红色=已排除
- 评分蓝点（最多 5 个）
- 选中帧蓝色高亮 + 轻微放大
- 自动滚动定位选中帧（smooth scroll）
- 底部图例 + 键盘快捷键提示

### `FrameEditor.tsx`
- 占据主要屏幕空间，大图预览
- 顶部工具栏：帧编号/文件名、状态选择器、评分星级、调整面板开关、上下帧导航
- 右侧面板：元数据（光圈/快门/日期）、帧备注（内联编辑）、调整参数（曝光/对比度/高光/阴影）
- 调整滑块为自定义样式，双向偏移可视化，非零值显示橙色
- 悬停图片区域有轻微缩放反馈

### `index.tsx`（主入口）
- 顶部导航：返回链接、品牌 Logo、阶段条、统计摘要、导出按钮、键盘帮助
- 统一的键盘事件监听（仅在 timeline 阶段激活，不干扰输入框）
- 导出视图内嵌进度模拟，完成后展示成功状态

## 键盘快捷键（时间轴阶段）

| 按键 | 操作 |
|------|------|
| `←` / `→` | 切换帧 |
| `1` – `5` | 设置评分（再次按同一键清除） |
| `Enter` | 切换已确认/已审阅 |
| `F` | 切换标记 |
| `R` | 切换排除 |
| `N` | 编辑备注（焦点在输入框时不触发） |
| `?` | 显示/关闭快捷键帮助面板 |

## 持久化

- `darkroom_last_roll_settings`：上次提交的卷信息，供"从上次复制"功能使用
- `darkroom_quickfill_draft`：QuickFillForm 的实时草稿，提交后清除

## 样式约定

- 主色：`blue-500` (#3b82f6)
- 成功：`green-500`
- 警告/标记：`amber-400/500`
- 排除：`red-500`
- 背景：白色/`gray-50`，高信息密度
- 过渡：`duration-100` – `duration-150`，快速直接
- 字体：系统 sans-serif + `font-mono` 用于数字/文件名
